# Matches-Simulador

Esse app simula placares dos jogos do Brasil, os placares são gerados conforme a força do time, podendo variar de 0 até 5 estrelas.

#### O que foi usado na criação do App:

 - Duas Activities
 - RecycleView
 - SwipeRefreshLayout
 - FloatingActionButton
 - CardView
 - Entre outras views.
